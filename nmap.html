Nmap — Network Scanner

What is Nmap?

Nmap is a tool used to scan networks and discover what devices and services are running. This process is called network scanning.

What Are Ports?

Ports are like doors into a system. Each port is used by a specific service.

If a port is open, it means something is running and accepting connections. This could be useful for legitimate access — or a potential entry point for an attacker.

Command Example

nmap -sV 192.168.1.0/24

-sV = detects service versions
192.168.1.0/24 = scans all devices on the network

Sample Output

22/tcp open ssh 80/tcp open http

Port 22 (SSH) → remote access service
Port 80 (HTTP) → web server

If these ports are open, it means those services are running and could be tested for vulnerabilities during a penetration test.

Why This Matters

Nmap helps security testers map out a target system. Knowing which ports are open shows where to focus further testing.
