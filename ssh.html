SSH — Secure Shell

What is SSH?

SSH (Secure Shell) is a protocol that allows you to connect to another computer over a network securely. It encrypts the connection so data cannot be easily intercepted.

Before SSH, tools like Telnet sent data in plain text, which made it easy for attackers to steal information. SSH solves this problem by using encryption.

What You Can Do with SSH

Once connected, you can control the remote system as if you were sitting in front of it.

  • Run commands
  • Access files
  • Manage system settings

Command Example

ssh student@192.168.1.100

ssh = the protocol used for secure remote access
student = username on the target system
192.168.1.100 = IP address of the target machine

Why This Matters in Ethical Hacking

During a penetration test, a tester might use a tool like Nmap to find a system, then connect to it using SSH if credentials are known or discovered.

This allows the tester to investigate the system from the inside.
