Brute-Force Attacks

What is a Brute-Force Attack?

A brute-force attack is a method where every possible password combination is tried until the correct one is found.

Brute Force vs Dictionary Attack

A brute-force attack tries every possible combination, which can take a long time. A dictionary attack uses a list of common passwords, making it faster but less thorough.

Example in a Penetration Test

During a penetration test, a tester might use a tool like Hydra to check if a system has weak passwords. If a simple password is found, it shows that the system is vulnerable.

Defenses

  • Account lockouts
  • Rate limiting
  • Multi-factor authentication

What a Finding Looks Like

If a tester successfully guesses a password, they would report it as a vulnerability. The risk is that an attacker could gain access to the system. The recommendation would be to enforce stronger passwords and add protections like MFA.
