Wireshark — Packet Analyzer

What is a Packet?

A packet is a small piece of data sent across a network. You can think of it like a letter in the mail. Each packet contains information like who sent it, where it’s going, and the actual message inside.

What Wireshark Does

Wireshark captures and displays network traffic in real time. It lets you see every packet moving across a network, which is useful for analyzing how systems communicate and spotting suspicious activity.

Why This Matters in Security

During a security investigation, Wireshark can reveal sensitive data being transmitted over the network. If traffic is not encrypted, an attacker could read it.

Example: Unencrypted Login

If a user logs into a website using HTTP instead of HTTPS, their username and password are sent in plain text. Wireshark can capture that packet and display the login information.

This is why secure websites use HTTPS — it encrypts the data so even if someone captures the packet, they cannot read the contents.
