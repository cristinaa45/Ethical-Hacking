Wireshark — Packet Analyzer

What is a Packet?

A packet is a small piece of data sent across a network. Think of it like a digital envelope carrying information from one computer to another.

What Wireshark Does

Wireshark lets you capture and inspect network traffic in real time. You can see exactly what devices are sending and receiving.

Security Use

During a security test, Wireshark can reveal passwords, messages, or sensitive data if they are not encrypted.

Example: HTTP Login

If a user logs in using HTTP (not HTTPS), the login data is sent in plain text. An attacker could capture it and read the username and password.
