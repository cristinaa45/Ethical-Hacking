Hydra — Credential Brute-Force Tool

What Hydra Does

Hydra is a tool used to test usernames and passwords by trying many combinations automatically. Security testers use it to find weak credentials on systems they are authorized to test.

What is a Wordlist?

A wordlist is a file that contains a list of possible passwords. Hydra uses this list to try each password one by one until it finds the correct one.

Example Command

hydra -l admin -P rockyou.txt ssh://192.168.1.50

-l admin → the username being tested
-P rockyou.txt → the list of passwords to try
ssh://192.168.1.50 → the target system and protocol

If Hydra finds the correct password, it will display it in the output.

⚠️ Ethics and Legality

Running Hydra against a system without permission is illegal. It is considered unauthorized access and can lead to serious consequences, including fines or criminal charges.

Ethical hackers only use Hydra when they have explicit permission from the system owner.

Defenses Against Brute-Force Attacks

  • Account lockouts after multiple failed attempts
  • Rate limiting to slow down login attempts
  • Multi-factor authentication (MFA)
