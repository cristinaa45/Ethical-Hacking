Hydra — Brute Force Tool

What Hydra Does

Hydra is used to test usernames and passwords automatically. It helps security testers find weak credentials.

Wordlists

A wordlist is a file of possible passwords. Hydra tries each one until it finds the correct one.

Command Example

hydra -l admin -P rockyou.txt ssh://192.168.1.50

-l = username
-P = password list
ssh:// = protocol + target

⚠️ Ethics (IMPORTANT)

Using Hydra without permission is illegal. It counts as unauthorized access and can lead to criminal charges.

Ethical hackers only run Hydra when they are authorized.

Defenses

  • Account lockouts
  • Rate limiting
  • Multi-factor authentication
